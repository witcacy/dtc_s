Reporte Detallado del DTC: U2FD9-7C

DTC Base: U2FD9

Status Byte: 7C (Hex) = 01111100 (Binario)

Este status byte indica qué tipo de fallo está registrado, si es confirmado, pendiente, si la luz MIL está encendida, etc.

Desglose del Status Byte

Bit Valor Descripción
7 0 Bit 7 - Test failed
6 1 Bit 6 - Test failed this monitoring cycle
5 1 Bit 5 - Pending DTC
4 1 Bit 4 - Confirmed DTC
3 1 Bit 3 - Test not completed this monitoring cycle
2 1 Bit 2 - Test failed since last clear
1 0 Bit 1 - Test not completed since last clear
0 0 Bit 0 - Warning Indicator Requested (MIL)
